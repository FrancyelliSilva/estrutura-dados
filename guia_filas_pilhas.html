Pilha Stack

Uma pilha é uma estrutura de dados linear que segue o princípio LIFO — Last In, First Out — onde o último elemento inserido é o primeiro a ser removido.

🍽️
Pilha de pratos
Você empilha pratos e sempre pega o do topo — o último colocado é o primeiro retirado.
↩️
Ctrl + Z
Desfazer ações em editores: a última ação realizada é a primeira a ser desfeita.
📚
Histórico do navegador
Cada página visitada é empilhada. O botão "voltar" remove do topo.

Visualização

← push / pop aqui (topo)
"C"  ← topo
"B"
"A"

Inserção e remoção sempre pelo topo

O elemento destacado em cima é sempre o topo da pilha — o único ponto de acesso. Não é possível remover "A" sem antes remover "C" e "B".

Implementação em Python

Python não tem um tipo nativo "pilha", mas a lista (list) funciona perfeitamente com os métodos append() e pop().

python 
# Criando e usando uma pilha com lista
pilha = []

# push — adiciona no topo
pilha.append("A")   # pilha: ["A"]
pilha.append("B")   # pilha: ["A", "B"]
pilha.append("C")   # pilha: ["A", "B", "C"]

# peek — ver o topo sem remover
topo = pilha[-1]    # topo: "C"

# pop — remove do topo
removido = pilha.pop()  # removido: "C", pilha: ["A", "B"]

# Verificar se está vazia
if not pilha:
    print("Pilha vazia!")

💡 Por que funciona? O método append() sempre adiciona ao final da lista (que representa o topo) e pop() sem argumento remove exatamente o último elemento — ambas operações em O(1).

Implementação com collections.deque

Para uso em produção, deque é preferível por ser thread-safe e ter desempenho consistente:

python 
from collections import deque

pilha = deque()

pilha.append(10)   # push
pilha.append(20)
pilha.append(30)

valor = pilha.pop()  # remove 30 (LIFO)
print(valor)         # 30

Exemplo prático: verificar parênteses balanceados

Um dos usos clássicos de pilha — verificar se uma expressão matemática tem parênteses corretamente fechados:

python 
def balanceado(expressao):
    pilha = []
    abertura = "({["
    fechamento = ")}" + "]"
    pares = {")": "(", "}": "{", "]": "["}

    for char in expressao:
        if char in abertura:
            pilha.append(char)       # empilha abertura
        elif char in fechamento:
            if not pilha or pilha[-1] != pares[char]:
                return False           # fechamento sem par
            pilha.pop()              # encontrou par, remove

    return len(pilha) == 0         # pilha vazia = tudo fechado

print(balanceado("(a + [b * c])"))  # True
print(balanceado("(a + [b * c)"))   # False

Fila Queue

Uma fila é uma estrutura linear que segue o princípio FIFO — First In, First Out — o primeiro elemento inserido é o primeiro a ser removido.

🏦
Fila do banco
Quem chega primeiro, é atendido primeiro. Novos clientes entram pelo final.
🖨️
Fila de impressão
O primeiro documento enviado é o primeiro impresso. Novos documentos vão para o final.
🌐
Pacotes de rede
Roteadores processam pacotes na ordem em que chegam — o primeiro a chegar, parte primeiro.

Visualização

enqueue →
"D"
"C"
"B"
"A" ← frente
← dequeue

Inserção pela traseira, remoção pela frente

Implementação em Python

O módulo collections.deque é a forma recomendada para filas — eficiente tanto para inserção quanto remoção:

python 
from collections import deque

fila = deque()

# enqueue — adiciona no final da fila
fila.append("A")       # fila: deque(["A"])
fila.append("B")       # fila: deque(["A", "B"])
fila.append("C")       # fila: deque(["A", "B", "C"])

# peek — ver a frente sem remover
frente = fila[0]       # frente: "A"

# dequeue — remove da frente (FIFO)
removido = fila.popleft()  # removido: "A"
                             # fila: deque(["B", "C"])

# Tamanho da fila
print(len(fila))         # 2

⚠️ Atenção! Não use list.pop(0) para filas — essa operação é O(n) porque desloca todos os elementos. Use sempre deque.popleft(), que é O(1).

Usando queue.Queue (thread-safe)

Para programas concorrentes ou multithread, o módulo padrão queue oferece sincronização automática:

python 
import queue

fila = queue.Queue()

fila.put("tarefa_1")   # enqueue
fila.put("tarefa_2")
fila.put("tarefa_3")

while not fila.empty():
    tarefa = fila.get()   # dequeue — FIFO garantido
    print(f"Processando: {tarefa}")

Exemplo prático: BFS — Busca em Largura

Filas são fundamentais para o algoritmo BFS (Breadth-First Search), que percorre grafos nível por nível:

python 
from collections import deque

def bfs(grafo, inicio):
    visitados = set()
    fila = deque([inicio])
    visitados.add(inicio)
    ordem = []

    while fila:
        no = fila.popleft()         # processa o mais antigo
        ordem.append(no)

        for vizinho in grafo[no]:
            if vizinho not in visitados:
                visitados.add(vizinho)
                fila.append(vizinho)  # enfileira vizinhos

    return ordem

grafo = {"A": ["B", "C"], "B": ["D"], "C": ["D"], "D": []}
print(bfs(grafo, "A"))  # ["A", "B", "C", "D"]

Pilha vs Fila

Característica Pilha LIFO Fila FIFO
Princípio Último a entrar, primeiro a sair Primeiro a entrar, primeiro a sair
Inserção Sempre no topo Sempre no final (traseira)
Remoção Sempre do topo Sempre da frente
Acesso Apenas o topo é acessível Apenas a frente é acessível
Python nativo list, deque deque, queue.Queue
Operação principal append() / pop() append() / popleft()
Analogia Pilha de pratos, histórico de ações Fila de banco, fila de impressão

Aplicações reais

Pilhas são usadas em…

  • 📝 Desfazer/Refazer (Ctrl+Z)
  • 🌐 Histórico do navegador
  • ⚙️ Chamadas de funções (call stack)
  • 🧮 Avaliação de expressões
  • 🔍 DFS — Busca em Profundidade

Filas são usadas em…

  • 🖨️ Fila de impressão

  • 🖥️ Agendamento de processos
  • 🔄 Buffers de streaming
  • 🌍 Cache de requisições web

Exercícios práticos

Exercício 01 — Pilha

Implemente uma função inverter_string(s) que usa uma pilha para inverter uma string sem usar slicing ([::-1]). Por exemplo: inverter_string("hello") deve retornar "olleh".

Ver solução 
def inverter_string(s):
    pilha = list(s)   # cada char é empilhado
    resultado = []
    while pilha:
        resultado.append(pilha.pop())
    return "".join(resultado)

print(inverter_string("hello"))  # "olleh"

Cada caractere é empilhado e, ao desempilhar, a ordem é invertida — exatamente o comportamento LIFO.

Exercício 02 — Fila

Crie uma fila para simular um sistema de atendimento. A fila deve suportar: adicionar clientes (chegar(nome)), atender o próximo (atender()) e ver quem está na frente (proximo()).

Ver solução 
from collections import deque

class SistemaAtendimento:
    def __init__(self):
        self.fila = deque()

    def chegar(self, nome):
        self.fila.append(nome)
        print(f"{nome} entrou na fila. Posição: {len(self.fila)}")

    def atender(self):
        if not self.fila:
            print("Fila vazia!")
            return
        cliente = self.fila.popleft()
        print(f"Atendendo: {cliente}")

    def proximo(self):
        if self.fila:
            print(f"Próximo: {self.fila[0]}")
Exercício 03 — Teórico

Considere a sequência de operações abaixo em uma pilha inicialmente vazia. Qual é o conteúdo final da pilha, da base ao topo?

push(5) → push(3) → push(7) → pop() → push(1) → pop() → push(9)

Ver solução

push(5) → [5]
push(3) → [5, 3]
push(7) → [5, 3, 7]
pop() remove 7 → [5, 3]
push(1) → [5, 3, 1]
pop() remove 1 → [5, 3]
push(9) → [5, 3, 9]

Conteúdo final: base → 5, 3, topo → 9

Exercício 04 — Desafio

Implemente uma fila usando apenas duas pilhas. As operações enqueue(x) e dequeue() devem se comportar como uma fila FIFO. Dica: use duas listas como pilhas.

Ver solução 
class FilaComPilhas:
    def __init__(self):
        self.entrada = []   # pilha de entrada
        self.saida = []     # pilha de saída

    def enqueue(self, x):
        self.entrada.append(x)  # sempre na entrada

    def dequeue(self):
        if not self.saida:
            # transfere entrada → saída (inverte ordem)
            while self.entrada:
                self.saida.append(self.entrada.pop())
        if not self.saida:
            raise IndexError("Fila vazia")
        return self.saida.pop()   # FIFO garantido

Ao transferir todos os elementos da pilha de entrada para a de saída, a ordem se inverte — transformando o comportamento LIFO em FIFO.
